# GrowthToDo

Personal website and consulting practice of Gautham.

**Live site:** [growthtodo.com](https://growthtodo.com)

## About

GrowthToDo helps founders cut through noise, generate clarity, and turn it into a sequenced to-do list that actually grows the company.

This site serves as:
- **Consulting hub** - Book clarity sprints and PMF workshops
- **Blog** - Writing on product, strategy, distribution, and building internet-first businesses
- **Projects** - Showcase of active and past ventures (SellOnTube, The Second Design, etc.)
- **Academy** - Turpentine Academy workshops for builders

## Tech Stack

- [Astro](https://astro.build) - Static site generator
- [TailwindCSS](https://tailwindcss.com/) - Utility-first CSS
- [DaisyUI](https://daisyui.com/) - Component library with custom "linear" dark theme
